# E-Commerce Website

A modern, responsive front-end e-commerce website built with HTML, CSS, and JavaScript. This project showcases a fully functional online shopping interface with multiple pages including product listings, shopping cart, blog, and contact sections.

## ðŸš€ Installation and Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/SDRasanjana/E-Commerce.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd E-Commerce
   ```

3. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server for better experience:
   
   **Using Python:**
   ```bash
   python -m http.server 8000
   ```
   
   **Using Node.js (http-server):**
   ```bash
   npx http-server
   ```
   
   Then open `http://localhost:8000` in your browser

## ðŸ’» Usage

### Navigation
- Use the navigation bar to browse between different pages
- Click on the shopping bag icon to view your cart
- Mobile users can access the menu via the hamburger icon

### Shopping
- Browse products on the home page or shop page
- Click on products to view detailed information
- Add items to cart using the shopping cart icon
- View and manage your cart on the cart page

### Pages Overview

- **Home (index.html)**: Landing page with featured products, new arrivals, and promotional banners
- **Shop (shop.html)**: Complete product catalog with filters
- **Single Product (sproduct.html)**: Detailed product view with specifications
- **Cart (cart.html)**: Shopping cart management
- **Blog (blog.html)**: Blog posts and articles
- **About (about.html)**: Company information and team details
- **Contact (contact.html)**: Contact form and information
